# Knowledge Representation and Reasoning in OWL  
## A Case Study on Movies

## Project Description
The project focuses on modeling a movie domain using a knowledge-based system.
An OWL ontology is created to represent movies, people, and genres, and
automated reasoning and query answering are demonstrated.

## Tools
- Protégé
- OWL 2
- OWL Reasoner
- SPARQL

## Repository Structure
- `ontology/` – OWL ontology file
- `queries/` – SPARQL queries
- `screenshots/` – screenshots of Protégé and reasoning results
